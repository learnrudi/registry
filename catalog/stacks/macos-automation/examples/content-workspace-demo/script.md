# Script Draft

## Hook

What if the agent did not just answer in chat, but helped set up the actual Mac
workspace you need to do the work?

## Beat 1

The stack can see the current Mac context: frontmost app, open windows, selected
Finder files, and available Shortcuts.

## Beat 2

It can prepare the local work surface: open the right app, reveal a project
folder, trigger a Shortcut, and create a Reminder for the next step.

## Beat 3

The important part is the boundary. The stack exposes typed actions, not raw
AppleScript. Risky writes are dry-run until explicitly confirmed.

## Close

This is the base layer for business, content, and productivity recipes that run
on the actual Mac instead of living only in the chat thread.
